TActiveDatePickerClientScript class.

Client-side date picker event OnDateChanged can be modified through the TActiveDatePicker::getClientSide property of a date picker.

The <tt>OnDateChanged</tt> event is raise when the date picker's date is changed. The formatted date according to TDatePicker::getDateFormat is sent as parameter to this event
