TActiveRecordInvalidFinderResult class.

TActiveRecordInvalidFinderResult defines the enumerable type for possible results if an invalid TActiveRecord::__call invoked.

The following enumerable values are defined:

  • Null: return null (default)
  • Exception: throws a TActiveRecordException
